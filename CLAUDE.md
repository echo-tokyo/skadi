личный кабинет для IT-школы.



Суть проекта:

\- Ученики: смотрят ДЗ в дашборде, прикрепляют решения

\- Преподаватели: создают задания, оценивают работы

\- Админ: создает группы и пользователей



Технологии: React + TypeScript + Redux Toolkit, архитектура FSD (Feature-Sliced Design), профессиональная реализация с современными паттернами.
